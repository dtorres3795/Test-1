Frequently Asked Questions

How long should a password be?

Ideally, a password should include at least 16 characters but including more can be a smart move. The longer the better! Remember that length is one of the most important factors in password strength.

Should I reuse passwords?

Never. If one account is breached, reusing passwords means all your other accounts become vulnerable too.

How often should I change my password?

Realistically, you should only really change a password if you suspect it has been compromised or if a service you use has been breached. Tho, it wouldnt hurt to be extra secure.

What makes a password weak?

Short length, common words, and predictable patterns like "123456" or "password", are great examples of what a weak password looks like. Try using reliable random passoword generators (like this website) to generate unpredictable and reliable passwords..
